PROPERLY FORMATTING YOUR .CSV or .TSV

To make the import process as accurate as poosible, we ask that you properly format your spreadsheet for upload to the BottleTrade Network.

Columns should be in order as follows:

Brewery Name - Beer Name - Size - Quantity - Date - Notes

*Note - Only Brewery and Beer are required fields. Quantity will default to 1 if no quantity is entered.

Please remove any generic words from the Brewery and Beer fields such as the, brewery, beer, company, ale, etc.

Columns should be in order as follows:

Brewery Name - Beer Name

*Note - Only Brewery and Beer are required fields.

Please remove any generic words from the Brewery and Beer fields such as the, brewery, beer, company, ale, etc.

Columns should be in order as follows:

Brewery Name - Description

*Note - Only Brewery are required fields.

Please remove any generic words from the Brewery field such as the, brewery, company, ale, etc.

Columns should be in order as follows:

Winery Name - Wine Name

*Note - Only Winery and Wine are required fields.

Please remove any generic words from the Winery and Wine fields such as the, winery, wine, company, etc.

Columns should be in order as follows:

Winery Name - Description

*Note - Only Winery are required fields.

Please remove any generic words from the Winery field such as the, winery, company, etc.

Columns should be in order as follows:

Distillery Name - Spirit Name

*Note - Only Distillery and Spirit are required fields.

Please remove any generic words from the Distillery and Spirit fields such as the, distillery, spirit, company, etc.

Columns should be in order as follows:

Distillery Name - Description

*Note - Only Distillery are required fields.

Please remove any generic words from the Distillery field such as the, distillery, company, etc.
